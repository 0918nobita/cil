# CIL

Common Intermediate Language

|Path|Description|
|:---|:---|
| `prog1.il` | Do nothing |
| `prog2.il` | Print `Hello, world!` |
| `hello.il` | Print `Hello, world!` |
| `addInt32.il` |  Sum up int32 values |
| `strIndexOf.il` | Call `String::indexOf(string)` |
| `cmdArgs.il` | Print command line arguments |
